# Team-Roster-Generator

GitHub Link : https://github.com/TheBryan-Acosta/Team-Roster-Generator

## **Description**

Challenge 10 This Weeks challenge was to create a CLI based HTML generator

### **What was developed**

A fully customizable professional team roster generator
-The user will be able to navigate through a simple menu to add as many employees as they wish
-This application will create a HTML file including each employee and their details on the page. 


### **Development** -

Inquirer was installed to allow users to input employee data, to later generate a webpage featuring the employees entered.
fs module for our file writing and copying.
jest for testing our employee contructors.

A sample of the webpage is contained inside the dist folder already.
